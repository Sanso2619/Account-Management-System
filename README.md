# C Account Management System

This is a simple C project for managing user accounts using file handling.

## Features
- Create a new account
- Login with username and password
- Recover forgotten username or password
- Change password
- Delete account
- Password masking for security

## File Structure
- `main.c` : main source code implementing all functionalities
- `.gitignore` : ignores unnecessary files like `.DS_Store` and build folders
- `account.txt` : stores user account data (not included in GitHub for privacy)
- `README.md` : project description

1. Compile the code using a C compiler:
